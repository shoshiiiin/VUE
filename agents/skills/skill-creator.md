# Skill Creator

## Beschreibung

Ein Skill, der dabei hilft, neue Agenten-Skills in einem standardisierten Format zu erstellen, um Arbeitsabläufe zu automatisieren [6], [5].

## Wann verwenden

Verwende diesen Skill, wenn der Benutzer einen wiederkehrenden Workflow automatisieren möchte oder explizit "Erstelle einen Skill für..." sagt [4].

## Workflow

1. **Analyse:** Verstehe das Ziel des neuen Skills basierend auf der Benutzeranfrage.
2. **Strukturierung:** Erstelle eine Markdown-Datei mit den folgenden Sektionen [5]:
   - **Beschreibung:** Was macht der Skill?
   - **Timing/Trigger:** Wann soll der Agent diesen Skill nutzen?
   - **Schritt-für-Schritt-Workflow:** Die exakten Anweisungen.
   - **Ressourcen:** Referenzen zu Skripten, Templates oder Dokumentationen (z. B. `agents.md`).
3. **Erstellung:** Lege die Datei im entsprechenden Skills-Ordner an (z. B. `agents/skills/`) [3].

## Richtlinien

- Nutze klare, direkte Sprache.
- Referenziere existierende Projektdateien, wo nötig.
